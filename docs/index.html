# Başlarken

Bu paketi kullanarak GİB eArşiv portal için fatura/makbuz oluşturabilir, portal üzerinde kayıtlı bilgi ve belgelere ulaşabilirsiniz.

# Kurulum

🛠️ Paketi composer ile projenize dahil edin;

composer require mlevent/fatura

# Api Bağlantısı

🔌 Gib eArşiv portala tarayıcınızdan bağlanmak için aşağıdaki linkleri kullanabilirsiniz;

  • https://earsivportaltest.efatura.gov.tr/login.jspopen in new window
  • https://earsivportal.efatura.gov.tr/intragiris.htmlopen in new window

# Test Kullanıcısı

GİB portal üzerinden test kullanıcısı oluşturmak ve token almak için;

use Mlevent\Fatura\Gib;

$gib = new Gib;

// Test kullanıcısına ait bilgileri getirir ve token alır
$token = $gib->setTestCredentials()
             ->login();

// Token
print_r($gib->getToken());

// Kullanıcı bilgileri
print_r($gib->getCredentials());





 
 






# Gerçek Kullanıcı

GİB portal bilgilerinizi kullanarak token almak için;

use Mlevent\Fatura\Gib;

$gib = new Gib;

// Kullanıcı bilgileriyle token alır
$token = $gib->setCredentials('3556988', '******')
             ->login();





 
 

Kullanıcı bilgilerini doğrudan login metoduna da iletebilirsiniz;

$token = $gib->login('3556988', '******');
 

Not: Token değerini herhangi bir durumda kullanmanız gerekmeyecek.

# Oturum Sonlandırma

Oturumun zaman aşımına uğramaması için işlemlerden sonra oturumu sonlandırın.

$gib->logout();
 

# Belge Listeleme

🔍 Portal üzerinde bulunan belgeleri, tarih aralığı belirtmek koşuluyla farklı şekillerde listeleyebilirsiniz. Liste içerisinde tüm belge tipleri bulunabilir.

$documents = $gib->getAll('01/09/2022', '15/09/2022');
 

Bu örnek, aşağıdaki şu diziyi döndürecektir;

Array
(
    [0] => Array

# Filtreleme

Kayıtları zincirleme metodlar kullanarak kolayca filtreleyebilirsiniz.

MetodAçıklama
onlyInvoice()Faturalar.
onlyProducerReceipt()Müstahsil makbuzları.
onlySelfEmployedReceipt()Serbest meslek makbuzları.
onlySigned()İmzalanmış belgeler.
onlyUnSigned()İmzalanmamış belgeler.
onlyDeleted()Silinmiş belgeler.
findRecipientName(string $value)Alıcı adına göre filtreleme.
findRecipientId(string $value)Alıcı vergi numarasına göre filtreleme.
findDocumentId(string $value)Belge numarasına göre filtreleme.
findEttn(string $value)Uuid numarasına göre filtreleme.
setLimit(int $limit, int $offset)Sonuçlar için limit belirleme.
sortAsc()Önce ilk kayıtlar.
sortDesc()(Varsayılan) Önce son kayıtlar.

# Belge Oluşturma

Oluşturulan belgeler, daha sonra imzalanmak üzere eArşiv portalda taslaklara kaydedilir. Kütüphaneyi kullanarak aşağıdaki belgeleri oluşturabilirsiniz;

  • Fatura
  • Müstahsil Makbuzu
  • Serbest Meslek Makbuzu

# Fatura

# Satış

Belge oluştururken named arguments (adlandırılmış değişkenler) veya dizilerle çalışabilirsiniz. faturaUuid ve belgeNumarasi belirtildiyse; portalda bu bilgilerle eşleşen belge güncellenir, diğer durumda portal üzerinde yeni bir belge oluşturulur.

use Mlevent\Fatura\Fatura;
use Mlevent\Fatura\Enums\InvoiceType;
use Mlevent\Fatura\Enums\Tax;
use Mlevent\Fatura\Enums\Unit;
use Mlevent\Fatura\Enums\Currency;
use Mlevent\Fatura\Models\InvoiceModel;
use Mlevent\Fatura\Models\InvoiceItemModel;

// Fatura detayları
$invoice = new InvoiceModel(
    tarih           : '20/10/2022',
    saat            : '23:50:48',
    paraBirimi      : Currency::USD,
    dovizKuru       : 18.56,
    faturaTipi      : InvoiceType::Satis,
    vknTckn         : '11111111111',
    aliciUnvan      : 'BISHOPOĞLU İNŞAAT MALZEMELERİ SAN. TİC. LTD. ŞTİ.',
    vergiDairesi    : 'Nilüfer Vd.',
    aliciAdi        : 'Walter',
    aliciSoyadi     : 'Bishopoğlu',
    mahalleSemtIlce : 'Nilüfer',
    sehir           : 'Bursa',
    ulke            : 'Türkiye',
);

// Ürün/Hizmetler
$invoice->addItem(
    new InvoiceItemModel(
        malHizmet  : 'Çimento',
        miktar     : 3,
        birim      : Unit::Ton,
        birimFiyat : 1259,
        kdvOrani   : 18,
    )
);

$gib = (new Gib)->login('333333054', '******');

if ($gib->createDraft($invoice)) {
    echo $invoice->getUuid(); // 04e17398-468d-11ed-b3cb-4ccc6ae28384
}

$gib->logout();

# Vergi Ekleme

💸 Oluşturulan belgeler, daha sonra imzalanmak üzere eArşiv portalda taslaklara kaydedilir. Kütüphaneyi kullanarak aşağıdaki belgeleri oluşturabilirsiniz;

$invoiceItem = new InvoiceItem(
    malHizmet : 'Çimento',
    miktar    : 3,
    birim     : Unit::Ton,
    birimFiyat: 1259,
    kdvOrani  : 18,
);

// Hizmete vergi ekleme
$invoiceItem->addTax(Tax::Damga,    rate: 15); // %15 damga vergisi
$invoiceItem->addTax(Tax::GVStopaj, rate: 25); // %25 gelir vergisi








 
 
 

Vergi listesine ulaşmak için Tax::cases() yöntemini kullanabilirsiniz;

// Vergiler
foreach (Tax::cases() as $tax) {
    echo $tax->name;  // Damga
    echo $tax->value; // 1047
    ...
}

// Vergi kodu kullanarak vergi ekleme
$invoiceItem->addTax(Tax::from(1047), rate: 15); // %15 damga vergisi
 
 
 
 
 
 

 
 

# Mükellef Bilgileri

TC Kimlik Numarası ya da Vergi Numarası ile mükellef sorgulamaya yarar. Fatura oluşturma aşamasında vergi numarasının doğruluğunu sorgulamak için kullanılabilir. Test ortamında sonuç boş döner.

$recipientData = $gib->getRecipientData('2920084496');

Bu örnek, aşağıdaki şu diziyi döndürecektir;

Array
(
    [unvan] => DENİZBANK ANONİM ŞİRKETİ
    [adi] =>
    [soyadi] =>
    [vergiDairesi] => Büyük Mükellefler VD. BAŞKANLIĞI
)

# GİB Profil Bilgileri

Portalda kayıtlı kullanıcı bilgilerine ulaşmak için getUserData metodunu kullanabilirsiniz. Sonuç bir dizi şeklinde döner.

$user = $gib->getUserData();

# Bilgileri Güncelleme

$updateUserData = $gib->updateUserData(
    UserDataModel::new(
        vknTckn : '3333333325',
        unvan   : 'XYZ Yazılım',
        ad      : 'Mert',
        soyad   : 'Levent',
        ...
    )
);

echo $updateUserData; // Bilgileriniz başarıyla güncellendi.

Önce portaldan profil bilgilerini okuyup daha sonra modele import ederek yalnızca belirli alanları güncelleyebilirsiniz.

$userData = UserDataModel::import($gib->getUserData());

$userData->apartmanAdi = 'Lale Apartmanı';
$userData->kapiNo      = '12';

echo $gib->updateUserData($userData); // Bilgileriniz başarıyla güncellendi.

# Birimler

# Ürün/Hizmet Birimleri

use Mlevent\Fatura\Enums\Unit;

foreach (Unit::cases() as $unit) {
    echo $unit->name;    // Dk
    echo $unit->value;   // D61
    echo $unit->alias(); // Dakika
}

# Vergi Birimleri

use Mlevent\Fatura\Enums\Tax;

foreach (Tax::cases() as $tax) {
    echo $tax->name;    // BankaMuameleleri
    echo $tax->value;   // 0021
    echo $tax->alias(); // Banka Muameleleri Vergisi
}

# Para Birimleri

use Mlevent\Fatura\Enums\Currency;

foreach (Currency::cases() as $currency) {
    echo $currency->name;    // TRY
    echo $currency->value;   // TRY
    echo $currency->alias(); // Türk Lirası
}
