# Vergi Ekleme

Oluşturulan belgeler, daha sonra imzalanmak üzere eArşiv portalda taslaklara kaydedilir. Kütüphaneyi kullanarak aşağıdaki belgeleri oluşturabilirsiniz;

$invoiceItem = new InvoiceItem(
    malHizmet : 'Çimento',
    miktar    : 3,
    birim     : Unit::Ton,
    birimFiyat: 1259,
    kdvOrani  : 18,
);

// Hizmete vergi ekleme
$invoiceItem->addTax(Tax::Damga,    rate: 15); // %15 damga vergisi
$invoiceItem->addTax(Tax::GVStopaj, rate: 25); // %25 gelir vergisi








 
 
 

Vergi listesine ulaşmak için Tax::cases() yöntemini kullanabilirsiniz;

// Vergiler
foreach (Tax::cases() as $tax) {
    echo $tax->name;  // Damga
    echo $tax->value; // 1047
    ...
}

// Vergi kodu kullanarak vergi ekleme
$invoiceItem->addTax(Tax::from(1047), rate: 15); // %15 damga vergisi
